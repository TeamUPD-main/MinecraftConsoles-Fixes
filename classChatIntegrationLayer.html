◆ OnOutgoingChatPacketReady()

void ChatIntegrationLayer::OnOutgoingChatPacketReady ( __in Microsoft::Xbox::GameChat::ChatPacketEventArgs^  args)

Send the chat packet to all connected consoles.

To integrate the Chat DLL in your game, change the following code to use your game's network layer. You will need to isolate chat messages to be unique from the rest of you game's other message types. When args->SendPacketToAllConnectedConsoles is true, your game should send the chat message to each connected console using the game's network layer. It should send the chat message with Reliable UDP if args->SendReliable is true. It should send the chat message in order (if that feature is available) if args->SendInOrder is true

Parameters
argsDescribes the packet to send
